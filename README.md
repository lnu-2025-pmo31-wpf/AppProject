# Finance Manager
Barabaliuk Vladislav and Miskiv Igor

## Опис проєкту
Finance Manager – це десктопна аплікація для обліку особистих фінансів.  
Вона дозволяє користувачам контролювати свої доходи та витрати, аналізувати фінансові потоки та формувати звіти для прийняття кращих рішень.

## Ціль аплікації
- Надати зручний інструмент для ведення особистого або сімейного бюджету.  
- Допомогти користувачам виявити, на що витрачаються кошти.  
- Формувати статистику для оптимізації витрат.

## Типи користувачів
- **Звичайний користувач** – веде власний бюджет.  
- **Сім’я / група** – кілька людей можуть вести спільний облік.

## Технології які будемо використовувати
- **C# .NET (WPF)** – для розробки десктопної аплікації.  
- **Entity Framework Core** – ORM для роботи з базою даних.  
- **ADO.NET** – виконання SQL-запитів для звітів.  
- **SQLite** – база даних (локальний файл).  
- (опціонально) **LiveCharts** – для побудови графіків і діаграм. 

## Основний функціонал
- Додавання / редагування / видалення транзакцій (доходи та витрати).  
- Категоризація витрат (їжа, транспорт, розваги, інше).  
- Пошук і фільтрація операцій за датою та категорією.  
- Автоматичний підрахунок балансу.  
- Статистика: щомісячні витрати, топ-категорії, діаграми. 

## Аналіз подібних аплікацій
### 1. Wallet (BudgetBakers)
- Зручний інтерфейс, синхронізація з банками.  
- ❌ Повна версія платна, є обмеження у безкоштовній.  

### 2. Spendee
- Красива візуалізація витрат, підтримка сімейних акаунтів.  
- ❌ Обмежений функціонал без підписки.  

### 3. Money Lover
-  Проста система категорій, нагадування про платежі.  
- ❌ Надокучлива реклама у безкоштовній версії.
